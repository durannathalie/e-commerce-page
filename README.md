# mi-primer-proyecto-movil
mi primer proyecto curso html-css

# Instalacion de proyecto github 
1.- Para clonar el repositorio debemos dirigirnos a https://github.com/durannathalie/mi-primer-proyecto-movil/tree/main y en un terminal dentro de la ruta mi-proyecto-movil ejecutar git clone url-del-repositorio.

2.- Realizar cambios en las fuentes y luego agregar cambios en mi local con git add .

3.- Comitear cambios con un mensaje segun el cambio realizado git commit -m "El cambio".

4.- Pushear los cambios a la rama con git push origin (rama).

# Descripcion del proyecto

Pagina web que muestra productos diversos del area de alimentacion natural, de forma amigable y responsiva.
